A semi-regular meetup for London JavaScript developers

Gin.JS is a meetup for the London JavaScript community.

There are no talks or set agendas. It's just a place to chat, where everyone just happens to be a JavaScript developer.

The first GinJS is running soon. If you'd like to attend, message me..
