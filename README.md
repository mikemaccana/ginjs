## Ticketing - Tito

### Update old event

 - Pick last event
 - Click *customize*
 - On 'Basics' change old event URL to be 'ginjssomedate-'

### Create new event

 - Go to 'dashboard'
 - Click 'duplicate' event icon (looks like paste icon)
 - Click 'customize'
 - In details update 'title', set 'date', and set 'event URL' to ginjs
